What is GitHub?

GitHub is a platform for hosting code, collaborating on projects, and version control. It allows developers to store their code online, track changes, and work together with others.

What is Git?

Git is a version control system that helps track changes in files and coordinates work among multiple people. It is the underlying technology used by GitHub.

What is GitHub Desktop?

GitHub Desktop is a graphical interface that makes it easier to use Git and GitHub on your computer. It helps you manage repositories, commits, branches, and more without using command-line tools.

How to Create a GitHub Account

  • Go to github.com
  • Click on "Sign up" and fill in your details (username, email, password).
  • Verify your email to activate your account.

How to Create a Repository

  • After logging in, click the "+" icon in the top right corner and select "New repository".
  • Give your repository a name and description.
  • Choose between public or private.
  • Click "Create repository".

How to Host on GitHub Pages

  • Go to your repository's settings.
  • Scroll down to the "Pages" section.
  • Choose a branch (usually main) and click "Save".
  • Your site will be published at a URL like https://aimannuruddeen.github.io/My-First-IT-Project-/
